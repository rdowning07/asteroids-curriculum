# 🚀 Asteroids

A from-scratch implementation of Asteroids in Python and Pygame.
Built as Lesson 2 of a structured game development curriculum — written by hand, not generated by AI.

---

## What This Is

This is not a tutorial clone. Every line of this code was written by the developer as part of a deliberate learning curriculum progressing from Pong → Asteroids → Donkey Kong → Mario Bros → Castlevania → Symphony of the Night.

The goal is first-principles understanding: knowing not just what the code does, but why each piece exists.

---

## What the Code Does

### `main.py`

The game entry point. Initializes pygame, creates the window, runs the game loop.

### `ship.py`

The player ship. A triangle drawn with `pygame.draw.polygon()` using calculated points. Rotates based on keyboard input.

**The game loop**
while running:
handle input      ← quit, keypresses
update state      ← rotate ship, move objects
screen.fill()     ← wipe canvas black
draw everything   ← ship, asteroids, bullets
display.flip()    ← swap back buffer to screen
dt = clock.tick() ← cap at 60fps, calculate delta time

## What Was Learned Building This

### Delta Time

Movement is scaled by how long the last frame took, not by frame count. This makes the game run at the same speed on any machine.

### Vectors

A vector is just an x and y treated as one thing. The ship's velocity is a vector. Every frame, velocity gets added to position. In Pong this was implicit. In Asteroids it's explicit.

### Trigonometry for Rotation

`cos(angle)` gives the x component of a direction. `sin(angle)` gives the y component. Together they convert an angle into an (x, y) direction vector. This is how the ship's triangle points are calculated every frame.

### Classes

A class is a blueprint. `Ship` is a class. `ship = Ship(640, 360)` creates one instance of that blueprint with its own state. Methods are functions that belong to a class and operate on its data via `self`.

### pygame.draw.polygon()

Takes a list of (x, y) points and fills the pixels between them. The ship's three points are recalculated every frame using the rotation formula so the triangle always faces the right direction.
